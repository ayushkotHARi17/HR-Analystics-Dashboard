# HR-Analystics-Dashboard
This project analyzes HR data to understand employee attrition and workforce trends using Power BI.

**Key Insights**

Attrition analysis by department and job role

Total employees, attrition rate, active workforce

Age and salary-based analysis

Interactive dashboard with filters

**Tools Used**

Power BI

Excel / CSV

**Learnings**

Data cleaning and transformation

Dashboard creation in Power BI

Extracting insights from data
